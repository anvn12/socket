# Tổng quát
Giống kiểu làm lại cái đống khi bấm ftp <ip>

Nó sổ ra một đống thứ, làm lại mấy đấy




# Các bước
1. Kết nối với server
	Nhập IP
	Nhập user, password
1. Giao diện ftp
1. chức năng cơ bản (trừ put, mput)
1. mput, put -> thêm 1 dòng system (clamscan,,,,,)
